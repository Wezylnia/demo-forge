# Examples

## Initialize a recipe

```bash
demoforge init
```

## Validate a recipe

```bash
demoforge validate .demoforge/demo.yml
```

## Run the sample demo

```bash
demoforge run samples/cli-basic/.demoforge/demo.yml
```

## Render from an existing session

```bash
demoforge render assets/demo/session.json --format html,markdown
```
